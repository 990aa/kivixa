# Changelog

All notable changes to the Kivixa project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-10

### Added
- Initial development of Kivixa
- Cross-platform notes application with drawing capabilities
- Support for multiple file formats (.kvx)
- File manager for organizing notes
- Theme customization options
- Multiple font support
- Canvas with drawing tools
- Image support in notes
- PDF export functionality

### Fixed
- Windows build symlink conflicts
- File locking issues on Windows in FileManager.writeFile
- Test suite cleanup - removed golden test files

### Changed
- Set app version to 1.0.0 (buildNumber: 100000)
- Improved test coverage (119 tests passing)

### Removed
- Golden screenshot tests
- Audio-related features (previously removed)
